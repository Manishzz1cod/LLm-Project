# Mini Project 1 - Sentiment Analysis on Product Reviews

This mini project focuses on performing **sentiment analysis** on customer product reviews.  
The goal is to classify reviews into **Positive, Neutral, and Negative** sentiments using Natural Language Processing (NLP) techniques.

## Project Overview
- Loaded product review dataset into a pandas DataFrame.
- Performed **data cleaning** and preprocessing.
- Analyzed data distribution of sentiments.
- Applied recommended evaluation metrics like **Macro F1-score, Precision, Recall, Confusion Matrix**.
- Prepared the dataset for machine learning modeling.
